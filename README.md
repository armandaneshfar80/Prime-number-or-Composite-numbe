# Prime-number-or-Composite-numbe
This simple Python program checks whether a given number is prime or composite.
It uses the modulo (%) operator to determine if a number has divisors other than 1 and itself.

این یک برنامه‌ی ساده‌ی پایتون است که بررسی می‌کند عدد وارد شده اول است یا مرکب. در این برنامه از عملگر باقیمانده (%) برای تشخیص بخش‌پذیری استفاده می‌شود.
